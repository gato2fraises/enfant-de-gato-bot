Politique de Confidentialité - Enfant de Gato Bot

Dernière mise à jour : 14 septembre 2025

1. Introduction

Cette politique de confidentialité explique comment "Enfant de Gato" (le "Bot") collecte, utilise et protège vos informations lorsque vous utilisez nos services sur Discord.

2. Informations que nous collectons

2.1 Données automatiquement collectées

  • Identifiants Discord : ID utilisateur, ID serveur, ID canal
  • Commandes utilisées : Historique des commandes pour le fonctionnement du bot
  • Messages de commandes : Contenu des commandes pour traitement
  • Données de serveur : Nom du serveur, nombre de membres (pour les statistiques)

2.2 Données de musique

  • URLs YouTube : Liens fournis pour la lecture de musique
  • Queue de musique : Liste temporaire des chansons en attente
  • Préférences audio : Volume, mode boucle (stockage temporaire)

2.3 Données de tickets

  • Contenu des tickets : Messages dans les canaux de support
  • Participants : Utilisateurs impliqués dans les tickets
  • Logs de modération : Actions effectuées par les modérateurs

3. Comment nous utilisons vos données

3.1 Fonctionnement du bot

  • Exécuter les commandes demandées
  • Maintenir les queues de musique
  • Gérer les systèmes de tickets
  • Fournir des statistiques de serveur

3.2 Amélioration du service

  • Analyser les erreurs et bugs
  • Optimiser les performances
  • Développer de nouvelles fonctionnalités

4. Stockage et rétention des données

4.1 Données temporaires

  • Queue de musique : Supprimée à l'arrêt du bot ou déconnexion
  • Cache de commandes : Supprimé après 24 heures
  • Sessions vocales : Aucune donnée persistante

4.2 Données persistantes

  • Configuration serveur : Conservée tant que le bot est présent
  • Logs d'erreur : Conservés 30 jours maximum
  • Statistiques anonymisées : Conservées indéfiniment

5. Partage des données

Nous ne vendons, ne louons, ni ne partageons vos données personnelles avec des tiers, sauf :

  • Services tiers nécessaires : YouTube API pour la musique
  • Obligations légales : Si requis par la loi
  • Protection des droits : Pour prévenir les abus

6. Sécurité des données

  • Chiffrement des connexions (HTTPS/WSS)
  • Accès restreint aux données
  • Surveillance des accès non autorisés
  • Mises à jour de sécurité régulières

7. Vos droits

7.1 Droits d'accès et de contrôle

  • Accès : Demander quelles données nous avons sur vous
  • Rectification : Corriger des données incorrectes
  • Suppression : Demander la suppression de vos données
  • Portabilité : Recevoir vos données dans un format lisible

7.2 Comment exercer vos droits

Pour exercer ces droits, contactez-nous via :

  • GitHub Issues : https://github.com/gato2fraises/enfant-de-gato-bot/issues
  • Discord : Contactez un administrateur du serveur

8. Services tiers

8.1 YouTube

Pour la fonctionnalité musique, nous utilisons l'API YouTube. Consultez la politique de confidentialité de Google.

8.2 Discord

Ce bot fonctionne sur la plateforme Discord. Consultez la politique de confidentialité de Discord.

9. Utilisateurs mineurs

Ce bot respecte les conditions d'âge de Discord (13 ans minimum). Nous ne collectons pas sciemment de données d'enfants de moins de 13 ans.

10. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications importantes seront communiquées via :

  • Mise à jour de la date en haut de cette page
  • Annonce dans les serveurs utilisant le bot
  • Notification sur notre repository GitHub

11. Contact

Pour toute question concernant cette politique de confidentialité :

  • GitHub : https://github.com/gato2fraises/enfant-de-gato-bot
  • Issues : Créer une issue
  • Discord : Contactez gato2fraise sur Discord

12. Conformité

Cette politique est conçue pour être conforme aux réglementations suivantes :

  • RGPD (Règlement Général sur la Protection des Données)
  • CCPA (California Consumer Privacy Act)
  • Discord Developer Policy
